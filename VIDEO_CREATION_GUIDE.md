### Phase 3: Build Scenes (30 mins)
For each scene in the script:

2. Add text overlay from script
3. Animate text (fade in)
4. Add Ken Burns zoom effect to image
5. Set timing per script

### Phase 4: Transitions (10 mins)

3. Keep transitions 0.5-1 second

### Phase 5: Audio (15 mins)
1. Add background music to entire timeline
2. Adjust volume (soft, background level)
3. Add voiceover/narration
4. Sync timing with visuals

### Phase 6: Review & Export (10 mins)
1. Watch full preview
2. Check timing against script
3. Adjust any issues
4. Export as MP4 (1080p quality)

**Total Time:** ~75 minutes for first video
